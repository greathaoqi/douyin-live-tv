# Douyin Live TV 使用指南

## 功能概览

| 功能 | 说明 |
|------|------|
| 抖音账号登录 | 登录后获取访问抖音直播数据权限 |
| 收藏直播间 | 保存常用直播间，方便快速访问 |
| 实时数据显示 | 观看人数、点赞数、礼物总值实时显示 |
| 直播视频播放 | 内置视频播放器预览直播 |
| 统计数据悬浮叠加 | 数据显示在视频上方，不遮挡画面 |
| 全屏/悬浮切换 | 可切换全屏视频或带数据悬浮 |
| 画中画播放 | 支持 PiP 画中画模式后台播放 |
| 手动刷新 | 下拉即可手动刷新数据 |
| 自动刷新 | 每30分钟自动刷新一次数据 |
| Top Shelf 快速访问 | Apple TV 主屏幕快捷访问收藏 |

## 首次登录

### 1. 获取抖音授权

1. 打开应用后，会进入登录页面
2. 使用手机扫描屏幕上的二维码
3. 在手机抖音中确认登录
4. 登录成功后会自动跳转到主界面

> 注意：登录令牌会安全存储在 Apple TV 的 Keychain 中，重启应用后不需要重新登录。令牌过期后会自动刷新。

## 添加收藏直播间

### 方法一：通过房间ID添加

1. 在主界面，点击 "添加直播间"
2. 输入抖音直播间 ID（可以从抖音分享链接中提取）
3. 点击 "确认添加"
4. 添加成功后会自动进入该直播间监控界面

### 方法二：通过分享链接添加

1. 在抖音App中打开目标直播间
2. 点击分享，选择 "复制链接"
3. 在 Douyin Live TV 的添加界面粘贴链接
4. 应用会自动提取房间ID并添加

> 提示：房间ID通常是数字，例如 `12345678`

## 监控直播数据

### 主界面说明

进入直播间后，你会看到：

- **顶部状态栏**：直播间名称、最后更新时间
- **中央区域**：直播视频预览播放
- **左上角悬浮数据面板**：
  - 👁️ **观看人数**：当前直播间在线观看人数
  - ❤️ **点赞数**：累计点赞数量
  - 🎁 **礼物总值**：累计礼物价值

### 数据刷新

- **手动刷新**：在当前界面向下拉（使用遥控器向下滑动）触发刷新
- **自动刷新**：应用会每30分钟自动刷新一次数据
- 后台刷新通过系统 `BackgroundTasks` 框架实现，省电高效

## 视频播放

### 基本操作

- **播放/暂停**：点击视频区域切换播放暂停
- **全屏模式**：点击菜单切换到全屏视频，隐藏统计数据
- **悬浮模式**：从全屏切换回悬浮模式，统计数据显示在左上角
- **画中画 (PiP)**：当你切换到其他应用时，视频会自动进入画中画模式继续播放

### 播放状态提示

- **直播中**：视频正常播放，数据正常显示
- **未开播**：显示 "当前未开播" 提示，数据保留上次刷新结果
- **加载失败**：显示错误提示，可手动重试

## 收藏管理

### 查看收藏列表

在应用主界面（标签页左侧）会显示所有已收藏的直播间，使用遥控器上下选择，点击进入。

### 删除收藏

1. 在收藏列表中，聚焦到要删除的直播间
2. 长按遥控器确认按钮（或点击菜单按钮）
3. 选择 "删除" 选项
4. 确认删除即可

### 快速启动

应用会记住你**最后一次查看的直播间**，下次启动应用会自动打开该直播间。

## Top Shelf 快速访问

应用启用了 Top Shelf 扩展：

1. 在 Apple TV 主屏幕，聚焦到 Douyin Live TV 应用图标
2. 上方会显示你的收藏直播间列表
3. 直接点击即可快速启动应用并进入对应直播间

这个功能让你无需进入应用列表导航，一步直达目标直播间。

## 自动刷新机制

### 工作原理

- 应用使用系统 `BackgroundTasks` 框架 scheduling 每30分钟一次的后台刷新
- 后台刷新会更新数据，下次打开应用时能看到最新数据
- 只有当应用在后台时才会触发，不耗电

### 注意事项

- 自动刷新不能保证精确到分钟，系统会根据使用情况优化
- 如果长时间不打开应用，系统可能会暂停后台刷新
- 打开应用后总会立即刷新一次数据

## 常见问题

### Q: 登录过期了怎么办？

**A:** 应用会自动处理令牌刷新，通常不需要用户干预。如果确实提示登录过期，退出应用重新打开会重新进入登录流程。

### Q: 视频无法播放？

**A:** 检查：
1. 确认直播间正在直播
2. 确认网络连接正常
3. 某些直播间可能有播放限制，这种情况下只能查看数据不能播放视频

### Q: 数据不更新？

**A:** 尝试：
1. 手动下拉刷新
2. 退出应用重新打开
3. 检查网络连接是否正常

### Q: 可以同时监控多个直播间吗？

**A:** v1.0 不支持同时监控多个，只能一次监控一个。切换直播间需要回到收藏列表选择。这是设计决定，保持应用简单。

### Q: 可以看弹幕/评论吗？

**A:** v1.0 不支持弹幕显示功能，只展示统计数据。项目目标就是专注于数据监控。

### Q: 应用闪退了怎么办？

**A:**
1. 尝试删除应用重新部署
2. 检查是否是最新代码
3. 查看 Xcode 控制台日志排查问题

### Q: 收藏的数据会丢失吗？

**A:** 收藏数据通过 SwiftData 保存在本地，不会丢失，除非你删除应用。重新部署应用只要不删除应用，数据会保留。

## 使用提示

1. **大屏观看体验**：建议坐在沙发距离电视2-3米使用，数据字体已经放大优化远观
2. **开机启动**：Apple TV 会记住最后打开的应用，开机后直接进入即可
3. **低功耗**：自动刷新不会消耗太多电量，因为使用系统原生框架

## 数据隐私

- 所有数据保存在本地，不会上传到任何第三方服务器
- 登录令牌仅存储在设备 Keychain 中
- 只和抖音 API 通信获取数据，没有其他数据收集
