Create `DouyinLiveTV/UI/WatchLive/LiveRoomViewModel.swift`:
- Import Combine, SwiftUI, AVKit
- `LiveRoomViewModel: ObservableObject`
- Injected dependencies: `playerService: PlayerService`, `liveStatsService: LiveStatsService`, `appLifecycleService: AppLifecycleService`
- `@Published private(set) var stats: LiveStats?`
- `@Published private(set) var showOverlay: Bool = true`
- `@Published private(set) var isLoading: Bool = false`
- `@Published private(set) var errorMessage: String?`
- `@Published var roomTitle: String = ""`
- Private `cancellables = Set<AnyCancellable>()`
- `func toggleOverlay()` → flips `showOverlay` value (LIVE-06 toggle requirement)
- `func loadRoom(roomId: String, streamURL: URL?) async` →:
  1. Sets isLoading = true, errorMessage = nil
  2. If streamURL exists, calls `playerService.loadVideo(url: streamURL)` and `playerService.play()`
  3. Calls `liveStatsService.fetchStats(for: roomId)` to get stats
  4. Updates `stats` on completion
  5. Sets isLoading = false, handles errors by setting `errorMessage`
- Subscribe to `appLifecycleService.$currentState`:
  - When .background → `playerService.pause()`
  - When .foreground → `playerService.resume()` (per UI contract interaction requirement)
- Store subscriptions in `cancellables`
