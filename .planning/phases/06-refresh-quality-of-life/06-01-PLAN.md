  <action>
Add a public async `refresh()` method to LiveRoomViewModel that:
1. Checks if there is a currently loaded room (stats != nil)
2. If no room is loaded, returns without doing anything (pull-to-refresh only enabled when room loaded per D-01)
3. Sets `isLoading = true` and clears `errorMessage`
4. Uses `liveStatsService.fetchStats(for:)` to fetch updated statistics
5. Updates the `stats` published property with new data
5. Updates `isLive` status
6. Sets `isLoading = false` on success
7. Catches errors, sets `errorMessage`, sets `isLoading = false` on failure

The method signature should be:
```swift
public func refresh() async
```

Keep the existing `loadRoom(roomId:streamURL:)` method unchanged - `refresh()` will reuse it when the room is already loaded. The room ID is available from the loaded `stats`.

If you need to store the current room ID, add a private property `currentRoomId: String?` to track it. Set it in `loadRoom` when a room loads successfully.
