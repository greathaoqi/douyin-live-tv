  <action>
1. Create new directory `DouyinLiveTVTopShelf/`
2. Create `DouyinLiveTVTopShelf/Info.plist` configured as a TV Top Shelf extension:
   - `NSExtension -> NSExtensionAttributes -> EXAppExtensionBrowseEnabled: false`
   - `NSExtension -> NSExtensionAttributes -> EXAppExtensionEnableQuickLook: false`
   - `NSExtension -> NSExtensionAttributes -> TVTopShelfContentKind: TopShelfSectioned`
   - `NSExtension -> NSExtensionPointIdentifier: com.apple.tv-top-shelf`
   - `NSExtension -> NSExtensionPrincipalClass: $(PRODUCT_MODULE_NAME).DouyinLiveTVTopShelf`

3. Create `DouyinLiveTVTopShelf/DouyinLiveTVTopShelf.swift` that implements:
   - Class `DouyinLiveTVTopShelf: TVTopShelfProvider`
   - `var topShelfContent: some TVTopShelfContent { get }`
   - In `topShelfContent`:
     - Initialize `ModelContainer` for `LiveRoom` (same as main app)
     - Fetch up to 4 favorites sorted by `lastViewed descending` (per D-27)
     - Create a single section with items
     - For each item, create `TVTopShelfItem` with:
       - Image: use placeholder system image "tv" (user can add custom later)
       - Title: room title
       - Subtitle: author nickname
       - Has badge: green badge if `isLive == true`
       - Use `userActivity` with `NSUserActivity(activityType: "com.douyinlivedtv.openRoom")`
       - Set `userActivity.userInfo["roomId"] = room.roomId`
     - Handle empty state: show message "No favorites added yet. Add a live room in the app to see it here."
     - Handle error state: show message "Could not load favorites. Open the app to refresh."

4. Follow the decision: maximum 4 items (D-26) in Top Shelf per Apple TV design guidelines.

Note: Because Xcode project modifications must be done manually, this task only creates the source files. The executor will instruct user to add the extension target in Xcode after file creation.
