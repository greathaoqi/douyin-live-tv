## Spacing Scale

Declared values (multiples of 4) from existing patterns:

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4pt | Icon gaps, inline spacing |
| sm | 8pt | Compact element spacing |
| md | 16pt | Default element spacing |
| lg | 24pt | Section spacing |
| xl | 32pt | Card/overlay padding, layout gaps |
| 2xl | 48pt | Major section breaks |
| 3xl | 64pt | Page-level spacing |

Exceptions: All focusable items must maintain minimum 88pt height per tvOS HIG (already established in Phase 3). Top Shelf items follow system-provided spacing defaults.

## Typography

Existing pattern from previous phases (confirmed for tvOS couch-distance readability):

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|

| Label | 28pt | regular (400) | 1.5 |

## Color

SwiftUI system colors already follow 60/30/10 contract:

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | systemBackground / black (dark mode) | Main background, full-screen surfaces |
| Secondary (30%) | secondarySystemBackground | Cards, cells, overlay backgrounds |
| Accent (10%) | accentColor (system green by default) | Live status indicators, focus indicators |
| Destructive | systemRed | Error messages |

Accent reserved for:
- Live status indicator dots (already used elsewhere, extend to Top Shelf)
- Focus effects (handled by system)
- Refresh progress indicator
