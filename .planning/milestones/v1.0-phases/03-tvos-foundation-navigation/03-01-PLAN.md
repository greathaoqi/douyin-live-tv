4. Create `AppLifecycleService.swift` with:
   - `class AppLifecycleService` that uses `NotificationCenter` to observe system events
   - `@Published private(set) var currentState: AppLifecycleState = .foreground`
   - Uses Combine to hold cancellables
   - On `didEnterBackgroundNotification`, sets `currentState = .background`
   - On `willEnterForegroundNotification`, sets `currentState = .foreground`
5. Create `AppLifecycleObserver.swift` in `UI/Common/`:
   - `class AppLifecycleObserver: ObservableObject`
   - `@Published var isInBackground: Bool = false`
   - Uses the injected `AppLifecycleService` to synchronize its published state
6. Update `DependencyContainer.swift`:
   - Add `public let appLifecycleService: AppLifecycleService`
   - Initialize it in `private init()` as `self.appLifecycleService = AppLifecycleService()`
7. Create unit test file `DouyinLiveTVTests/AppLifecycleTests.swift`:
   - Import XCTest and Combine
   - Test that initial state is .foreground
   - Test that posting didEnterBackgroundNotification changes state to .background
   - Test that posting willEnterForegroundNotification changes state back to .foreground
