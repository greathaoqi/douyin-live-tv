| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | systemBackground | Main background surfaces |
| Secondary (30%) | secondarySystemBackground | Cards, grouped content |

Accent reserved for:

- Focused login button highlight
- Activity indicators during login polling
- Error state accent highlighting

## Visual Hierarchy & Layout

**Primary focal point:** QR code display — this is the actionable center of the login screen. It should be prominent and take up sufficient space (minimum 280x280pt on tvOS).

**Reading order (visual flow):**
1.  **Heading** ("Douyin Live Login") — establishes context at the top
2.  **QR code** — primary interactive focal point in the center
3.  **Instructions** — explanatory text below the QR code
4.  **Primary CTA button** ("Start Login") — call-to-action at the bottom, focusable

**Layout structure:**
- Center-aligned vertically and horizontally on screen
- QR code container with accent color border
- All content fits within safe area (avoiding tvOS overscan regions)
- No scroll needed — all content fits on one screen for 1080p/4K displays

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Start Login |
| Empty state (not logged in) | Not logged in to Douyin |
| Empty state body | Scan the QR code with your Douyin mobile app to log in. |
| Error state (network error) | Could not connect to Douyin. Check your internet connection and try again. |
| Error state (login expired) | Your login session has expired. Please scan the QR code again to log in. |
| Error state (login failed) | Login process failed. Please generate a new QR code and try again. |
| Destructive confirmation | Log Out: Are you sure you want to log out? Your saved credentials will be removed. |

Notes:
- No destructive actions are required in this phase (logout can be deferred to later phases)
- If logout is included in this phase, use the confirmation dialog above
