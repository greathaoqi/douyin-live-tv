## End-to-End User Flows

All 6 main user flows are complete end-to-end:

1. **Login Flow:** User opens app → unauthenticated → shows LoginView → QR code displayed → user scans with phone → token stored → navigates to MainTabView ✓

2. **Add Room Flow:** User navigates to Add tab → enters room ID/URL → extracts ID → fetches stats → saves to SwiftData → navigates to Favorites ✓

3. **View Favorites Flow:** User selects Favorites tab → list of favorites appears (sorted by lastViewed descending) ✓

4. **Delete Favorite:** User enters edit mode → swipes to delete → confirms deletion → room removed from SwiftData ✓

5. **Select Favorite:** User taps favorite → updates lastViewed timestamp → navigates to Watch Live tab → stats loaded and displayed ✓

6. **App Launch with Last Room:** App launches → recalls last viewed room ID from UserDefaults → auto-navigates to Watch Live → auto-fetches stats ✓
